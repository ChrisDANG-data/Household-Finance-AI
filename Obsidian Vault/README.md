# FinIntel Obsidian vault

This folder is generated by **Household Financial Intelligence**.

## Open in Obsidian
1. Install [Obsidian](https://obsidian.md/).
2. **Open folder as vault** and select this directory.
3. Open **Graph view** (ribbon or Ctrl/Cmd+G) to visualize document ↔ category ↔ ledger links.

Re-sync from the FinIntel **Documents** page after new uploads.

Start at [[Household Index]].
